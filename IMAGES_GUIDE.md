## Recommandations Techniques

### Optimisation
- Compresser toutes les images pour le web
- Utiliser des formats modernes (WebP quand possible)
- Maintenir une qualité élevée tout en optimisant la taille

### Style Visuel
- **Palette de couleurs**: Cohérente avec le design (bleu #0D4BD3, jaune #FADD82)
- **Éclairage**: Lumineux et professionnel
- **Composition**: Moderne et dynamique
- **Personnes**: Diversité, professionnalisme, jeunesse

### Droits d'Usage
- S'assurer d'avoir les droits d'utilisation de toutes les images
- Privilégier les photos originales ou les banques d'images libres de droits
- Créditer les photographes si nécessaire

## Sources Recommandées
- **Unsplash**: Photos gratuites de haute qualité
- **Pexels**: Large sélection d'images professionnelles
- **Shutterstock**: Images premium pour un usage commercial
- **Photos originales**: Prises lors d'événements CMA pour l'authenticité
